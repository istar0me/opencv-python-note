# Face Recognition
* 參考：[OpenCV: Face Detection using Haar Cascades](https://docs.opencv.org/3.4.4/d7/d8b/tutorial_py_face_detection.html)
## Haar-like features（哈爾特徵）
* 人臉有對稱的特徵：眼睛、鼻子、耳朵等等
* 可以將人臉做二值化後，再來做判斷
* OpenCV 中有內建此方法

## Adaboost
